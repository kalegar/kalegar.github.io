Philips Hue Light Web App

Published: Mon 17 February 2025
By Andrew Smith

In Project.

In the summer of 2022 I recieved a couple Philips Hue smart bulbs as a gift. Of course I wanted to try out their API so got started on a web app to control my lights from my phone or tablet (or anywhere I could access a website from). I themed the app on Dungeons and Dragons as I intended to use it to create preset light scenes and effects I could create, swap between, and trigger easily through the app. You can find the sourcecode here, with the backend built using node and Vue for the frontend: https://github.com/kalegar/DnD-Hue
