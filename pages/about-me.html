About Me

Hello! My name is Andrew and I am yet another programmer with a git pages portfolio site!

The purpose of this site is to show off some of my own projects as well as my professional experience and interests.

One such interest is full stack development and I've worked on many projects in that vein both professionally and on my own time (check the articles here for some examples!)

I also have experience importing and converting data between different schemas and database engines, such as the following:

  • PostgreSQL
  • MySQL
  • SQLite
  • Firebird
  • Topspeed
  • SQL Server / MSSQL

I love to learn new technologies and techniques and am commited to staying up to date with the latest happenings in the software development world.

Recently I began taking some courses to further my database management and data analytics knowledge in areas such as:

  • Data manipulation in Python using Pandas
  • R programming language
  • Microsoft Power BI
