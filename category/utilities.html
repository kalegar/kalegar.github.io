Python Delphi dfm parser

Published: Mon 17 February 2025
By Andrew Smith

In Utilities.

Here's a small script I was working on to help automate updating field sizes on query objects in my Delphi programs. It parses a given Delphi dfm file or a directory of dfm files and updates any size fields on objects matching a list to a new size. I frequently modify this script to update other fields as needed and it has been handy in parsing out captions and text from controls for our translation project. https://github.com/kalegar/dfm-parser
