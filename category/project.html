A simple to use Minecraft administration plugin

Published: Mon 17 February 2025
By Andrew Smith

In Project.

I've been hosting Minecraft servers for my friends and family on and off since the game first came out and have built quite a few Java plugins for Bukkit, Spigot, Paper, etc etc. One that I've consistently kept up to date whenever I get back in to playing is an administration plugin I called AmavyaAdmin. It's a plugin that adds a number of useful features for running a minecraft server such as teleport requests, allowing players to set their own "home" teleports, a "Message of the Day", server rules, etc. Take a look at the sourcecode, and feel free to build it yourself if you'd like to try it out on your own server: https://github.com/kalegar/AmavyaAdmin
