Local Shopping App in Node.js and Vue

Published: Mon 28 October 2024
By Andrew Smith

In Projects.

In my spare time in 2022 I was building a web application to help connect people in Edmonton Alberta to local shops near them. The project backend was built using Node.js, with a Vue frontent. I used a PostgreSQL database to hold data on the merchants / shops and their products, and implemented a search function with many filters, including being able to search by proximity using GPS location services. To display a map of nearby businesses or a map of your search results, I utilized the Google Maps API. The site is not currently live anywhere, but you can take a look at the codebase here: https://github.com/kalegar/thelocalog
